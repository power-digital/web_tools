Project

Required keys for a valid dbt_project.yml.

Default model config

Applied to all models under the project namespace. Leave blank to omit.

Model folders (optional)

Per-subfolder overrides, e.g. staging as views, marts as tables.

Seeds (optional)

Config for CSV files in your seeds/ directory.

Seed folders (optional)

Per-subfolder overrides under seeds/.

Variables (optional)

Project-level vars available to all models. Values accept JSON for lists, dicts, numbers, booleans, and null — e.g. ["a","b"] or {"k":"v"}; bare text stays a string.

Paths

Comma-separated. Blank uses the dbt default.

Quoting

Leave on default to omit the key entirely.

Hooks

One statement per line.
