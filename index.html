Tools & Utilities

A collection of Power Digital tools and utilities.

  • DataDeveloper

    dbt Config Generator

    Fill out a guided form to generate dbt_project.yml, sources.yml, and packages.yml, with live YAML preview. Copy a link to save or share, or download the files.
  • DataDeveloper

    Cron Builder

    Build and read cron expressions with a visual editor, presets, or direct field input. See a plain-English description and the next eight run times. Handy for dbt Cloud and scheduled jobs.
  • Developer

    Regex Builder

    Describe what you want to match by stacking plain-English conditions — contains, starts/ends with, a number, one of these words, common formats — and the regular expression builds itself. Live test panel with match highlighting and a plain-English explanation. No AI, runs entirely client-side.
  • Developer

    Code Formatter

    Paste SQL, JSON, or Python and get it cleanly formatted — sqlfmt-style SQL with dialect support, pretty-printed JSON, and normalized Python indentation. Runs entirely client-side; nothing leaves your browser.

    Meeting Agenda

    Plan a meeting, take notes against each agenda item, time-box topics, and export or share when you're done.
